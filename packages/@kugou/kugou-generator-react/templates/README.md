### 开发者工具

利用开发者工具的远程调试功能，可以加速调试设备中的页面，下面简单介绍如何启用开发者工具

- 首先，若没有全局安装```react-devtools```需要先安装

```bash
npm i -g react-devtools
# or
yarn global add react-devtools
```

- 其次，在```.env```配置环境变量REACT_APP_DEVTOOL=true

- 最后，在终端执行```react-devtools```命令启动开发者工具

```bash
react-devtools
```
